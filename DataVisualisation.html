#customer functions to calculate several means
def avg_length_sub_cust(city):
    
    subscribers_length_mean = city.loc[city['usertype'] == 'Subscriber', 'tripduration'].mean()
    customer_length_mean = city.loc[city['usertype'] == 'Customer', 'tripduration'].mean()
    
    if subscribers_length_mean > customer_length_mean:
        best = 'Subscribers'
    else:
        best = 'Customers'
        
    return subscribers_length_mean, customer_length_mean, best

What is the structure of the data set?¶

In this data set are 183412 fordgobike journeys with 16 dimensions. Out of these 16 dimensions 1 is a boolean type, 2 are datetime, 9 are numerical and 4 are object type.

What is the interest of this project?¶

This project interests lies in discovering and understanding if the duration of a trip is dependent on one of those dimensions

What variables or features will help support the investigation?¶

I think that the duration of trip will be dependent of the start and the end station. Also, I think that more crowded places will have more rides i.e. will log more duration. Moreover some other variables of interest will be of demographic nature such as gender, birthyear etc.
